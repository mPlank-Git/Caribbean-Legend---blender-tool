Terrain RGB Mask Painter – Quick Guide

General Concept
---------------
A single RGB mask controls 4 texture layers:

Black (0,0,0) – Base layer (bottom)
Red           – layer 1
Green         – layer 2 (overrides Red)
Blue          – layer 3 (overrides everything)
--------------------------------------------------
1. Select the terrain mesh.

2. Make sure the mesh has two UV channels:
   UV1 – for tiled terrain textures.
   UV2 – for the RGB mask.

3. Assign textures for all layers:
   Base Layer
   Red Layer
   Green Layer
   Blue Layer

   Each layer uses:
   Albedo
   Normal
   RMA

   RMA packing:
   R = Roughness
   G = Metallic
   B = AO

4. Create the mask:
   Set mask name.
   Set mask resolution.
   Click Create RGB Mask.

5. Build the preview material:
   Click Build / Update Material.

6. Assign the mask for painting:
   Click Assign Mask For Paint.

7. Switch Blender to Texture Paint mode.

8. Paint the mask:
   Base – paints black and returns to the Base layer.
   R – paints the Red layer.
   G – paints the Green layer.
   B – paints the Blue layer.

9. Optional:
   Use ADD Paint OFF for normal Mix painting.
   Use ADD Paint ON if you want R/G/B to be added without erasing existing channels.
   Base always paints in Mix mode.

10. Preview modes:
   Material – final blended terrain.
   Mask – raw RGB mask.
   R / G / B – individual channels.

11. Mask tools:
   Blur – softens transitions.
   Fill Base / R / G / B – fills the entire mask with selected layer.

12. Save the mask:
   Click Save Mask.
   Export it as PNG.

13. Engine setup:
   Import mask as Non-Color / Linear.
   Use UV2 for the mask.
   Use UV1 for tiled textures.
