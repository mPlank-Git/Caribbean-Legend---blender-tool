Terrain RGB Mask Painter – Quick Guide

Общая логика
------------
Используется одна RGB-маска для управления 4 слоями:

Black (0,0,0) – Base слой (низ)
Red           – слой 1
Green         – слой 2 (перекрывает Red)
Blue          – слой 3 (перекрывает всё)

Слои автоматически смешиваются:
- без грязи
- без просвечивания базы
- с мягкими переходами

--------------------------------------------------

Подготовка модели
-----------------
Объект должен иметь 2 UV канала:

UV1 – для тайлинга текстур
UV2 – для маски

--------------------------------------------------

Настройка слоёв
---------------
Для каждого слоя задаются:

- Albedo
- Normal
- RMA (R=Roughness, G=Metallic, B=AO)

Слои:
- Base
- Red
- Green
- Blue

--------------------------------------------------

Создание маски
--------------
1. Указать имя и разрешение
2. Нажать:
   Create RGB Mask

3. Назначить маску:
   Assign Mask For Paint

--------------------------------------------------

Покраска
--------
Работа в режиме Texture Paint

Кнопки:

Base – стирает (чёрный)
R    – красный слой
G    – зелёный слой
B    – синий слой

Важно:
Маска — обычная RGB-текстура.
Правильное поведение слоёв считается в материале, а не в текстуре.

--------------------------------------------------

Инструменты
-----------
Blur – размывает маску (мягкие переходы)

Fill:
Base / R / G / B – заливка всей маски выбранным слоем

--------------------------------------------------

Просмотр
--------
Material – итоговый результат
Mask     – RGB маска
R / G / B – отдельные каналы

--------------------------------------------------

Экспорт
-------
Save Mask

Сохраняет PNG маску для использования в движке.

--------------------------------------------------

Важно для движка
----------------
- Маску использовать как Non-Color (linear)
- UV2 = маска
- UV1 = тайлинг текстур

--------------------------------------------------

Итог
----
1 маска = 4 материала

Плюсы:
- меньше памяти
- меньше размер билда
- быстрый пайплайн
- гибкое смешивание
