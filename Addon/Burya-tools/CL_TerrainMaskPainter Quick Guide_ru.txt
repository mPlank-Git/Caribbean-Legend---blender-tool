Terrain RGB Mask Painter – Quick Guide

Общая логика
------------
Используется одна RGB-маска для управления 4 слоями:

Black (0,0,0) – Base слой (низ)
Red           – слой 1
Green         – слой 2 (перекрывает Red)
Blue          – слой 3 (перекрывает всё)
--------------------------------------------------
1. Выбери mesh террейна.

2. Убедись, что у объекта есть два UV-канала:
   UV1 – для тайлинга текстур.
   UV2 – для RGB-маски.

3. Назначь текстуры для всех слоёв:
   Base Layer
   Red Layer
   Green Layer
   Blue Layer

   Для каждого слоя используются:
   Albedo
   Normal
   RMA

   Упаковка RMA:
   R = Roughness
   G = Metallic
   B = AO

4. Создай маску:
   Укажи имя маски.
   Укажи разрешение маски.
   Нажми Create RGB Mask.

5. Собери материал предпросмотра:
   Нажми Build / Update Material.

6. Назначь маску для покраски:
   Нажми Assign Mask For Paint.

7. Перейди в режим Texture Paint.

8. Крась маску:
   Base – красит чёрным и возвращает базовый слой.
   R – красит Red layer.
   G – красит Green layer.
   B – красит Blue layer.

9. Дополнительно:
   Use ADD Paint OFF – обычная покраска в режиме Mix.
   Use ADD Paint ON – R/G/B добавляются поверх, не стирая существующие каналы.
   Base всегда красит в режиме Mix.

10. Режимы предпросмотра:
   Material – итоговый смешанный террейн.
   Mask – исходная RGB-маска.
   R / G / B – отдельные каналы.

11. Инструменты маски:
   Blur – смягчает переходы.
   Fill Base / R / G / B – заливает всю маску выбранным слоем.

12. Сохрани маску:
   Нажми Save Mask.
   Экспортируй её в PNG.

13. Настройка в движке:
   Маску импортировать как Non-Color / Linear.
   UV2 использовать для маски.
   UV1 использовать для тайлинга текстур.
